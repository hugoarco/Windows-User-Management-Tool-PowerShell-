# Script-Admnistraci-n-Usuarios-Powershell
Gestión de usuarios Windows mediante PowerShell.  Script de administración básica que permite crear, borrar y comprobar usuarios del sistema mediante un menú interactivo. Incluye validación de usuario, opciones de gestión y manejo básico de errores.  Proyecto realizado como práctica de automatización y administración de sistemas en SMR.
